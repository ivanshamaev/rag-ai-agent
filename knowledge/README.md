# Учебник: Основы LLM и embeddings

Это структурированный учебник по ключевым темам, связанным с большими языковыми моделями (LLM), эмбеддингами, retrieval-методами, RAG-архитектурами, инфраструктуре данных, DevOps и системному дизайну ML-платформ.

Файлы в этой директории:

- `01_llm.md` — Что такое LLM, архитектура трансформеров, attention.
- `02_embeddings.md` — Эмбеддинги: определение, получение, типы, сравнение.
- `03_retrieval.md` — Методы retrieval: dense vs sparse, гибриды.
- `04_rag_architecture.md` — Архитектура RAG-систем и компоненты.
- `05_rag_pipeline.md` — Пайплайн RAG: ingestion → chunking → embedding → retrieval → генерация.
- `06_reranking.md` — Переранжирование и reranker-модели.
- `07_eval_frameworks.md` — Фреймворки для оценки (RAGAS, TruLens и др.).
- `08_vector_dbs.md` — Векторные базы данных: Milvus, FAISS, Qdrant и принципы работы.
- `09_data_infra.md` — Инфраструктура данных: типы БД, ETL, best practices.
- `10_databases.md` — Реляционные и NoSQL-базы: особенности и масштабирование.
- `11_etl.md` — ETL-процессы, инструменты: Spark, Airflow и др.
- `12_containerization_devops.md` — Docker, CI/CD, мониторинг ML-сервисов.
- `13_system_design_ml.md` — Системный дизайн для ML-платформ и метрики качества.

Если хотите — могу добавить примеры кода, диаграммы архитектур или чек-листы для развёртывания.
